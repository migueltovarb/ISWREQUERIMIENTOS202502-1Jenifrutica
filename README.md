**Sistema de Gestión de Notas Académicas - Django**

Sistema completo de gestión académica que permite la administración de estudiantes, profesores, cursos, calificaciones y generación de reportes académicos.

#### Características principaleses:
- Sistema de roles (Administrador, Profesor, Estudiante)
- Gestión completa de cursos y calificaciones
- Generación de reportes académicos en PDF
- Sistema de notificaciones
- Perfiles personalizables por usuario
